### High-Level Blueprint & Core Mechanics

The secret to building this in $T = 2\text{ hours}$ is to focus on a single, shared state (the **Campaign Context**) that feeds into three "Quick Generators."

```plaintext
                    ┌────────────────────────────┐
                    │      Campaign Context      │
                    │ (Location, Level, History) │
                    └──────────────┬─────────────┘
                                   │ (Injected into Prompts)
         ┌─────────────────────────┼────────────────────────┐
         ▼                         ▼                        ▼
┌──────────────────┐      ┌──────────────────┐     ┌──────────────────┐
│ 1. Dynamic Twist │      │   2. Cozy Loot   │     │  3. NPC Dialogue │
│ (Unexpected      │      │ (Imbued with     │     │ (Voiced, secret  │
│  encounters)     │      │  story relevance)│     │  motives)        │
└──────────────────┘      └──────────────────┘     └──────────────────┘
```

#### The Formula for Controlled Randomness
Traditional tables use a flat distribution $P(X) = \frac{1}{N}$. Our AI model calculates a contextual probability based on party level, environment, and history:

$$\text{Encounter Suitability} \propto f(\text{Party Level}, \text{Environment Match}, \text{Story Momentum})$$

If the party of Level $5$ ($APL = 5$) is in a frozen cavern, the AI will automatically filter out desert sand-worms and low-threat CR $1/4$ goblins, generating instead an unexpected frost-gargoyle alignment with a mechanical Challenge Rating $CR \approx APL \pm 1$ ($4 \le CR \le 6$).

#### 2. The Campaign State Manager (20 to 45 mins)
Write a Python utility (`src/campaign_manager.py`) to manage game state:

```python
# src/campaign_manager.py
import json
import os

STATE_FILE = "data/campaign_state.json"

DEFAULT_STATE = {
    "party_level": 3,
    "current_location": "The Whispering Swamps",
    "tone": "Dark & Gritty",
    "recent_events": "Defeated the swamp-witch; found a glowing obsidian key.",
    "known_npcs": [
        {"name": "Bryn", "role": "Witch's captive", "secret": "Actually a cultist spy"}
    ]
}

def load_state():
    if not os.path.exists(STATE_FILE):
        return DEFAULT_STATE
    with open(STATE_FILE, "r") as f:
        return json.load(f)

def save_state(state):
    os.makedirs(os.path.dirname(STATE_FILE), exist_ok=True)
    with open(STATE_FILE, "w") as f:
        json.dump(state, f, indent=4)
```
